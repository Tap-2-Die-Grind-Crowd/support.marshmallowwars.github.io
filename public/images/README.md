# Background Image Setup

To complete the setup, please save your unicorn rainbow image as:

**Filename:** `unicorn-rainbow-bg.webp`  
**Location:** `/public/images/unicorn-rainbow-bg.webp`

The image will be used as the full-page background for the Marshmallow Wars support page.

## Supported formats

- JPG/JPEG (recommended for photos)
- PNG (for images with transparency)
- WebP (for better compression)

## Recommended specifications

- **Size:** 1920x1080 or larger
- **Format:** WebP for best compression and quality
- **Quality:** Optimized for web (80-90% quality)

Once you save the image with the correct filename, the background will automatically appear on your page!
