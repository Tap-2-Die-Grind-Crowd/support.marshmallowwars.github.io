## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm run export
```

This will create a static export in the `out` directory ready for GitHub Pages deployment.

## Customization

- Update the App Store URL in `pages/index.tsx`
- Modify the support email in the same file
- Customize colors and themes in the CSS-in-JS styles
- Replace SVG images in the `public/images/` directory

## Deployment

This project is configured for GitHub Pages deployment with static export. The build process creates optimized, compressed assets perfect for mobile users.
